# v5.1 验收清单

本清单用于确认 `v5.1` 是否已经达到“可验证、可诊断、可启用、可维护”的目标。

---

## 1. LLM provider 能力

- [ ] `.env` 中已配置默认 `LLM_PROVIDER`
- [ ] 至少一个 provider 凭据有效
- [ ] 如启用第二 provider，已配置 `OPENAI_COMPATIBLE_*` 变量
- [ ] 如需阶段路由，已配置 `LLM_<STAGE>_PROVIDER / MODEL`
- [ ] 如需基础治理，已配置 `LLM_<STAGE>_TIMEOUT_MS / MAX_RETRIES`
- [ ] `novel doctor` 能展示默认 provider、阶段路由、timeout、retry
- [ ] `novel regression run llm-provider-smoke` 通过
- [ ] `novel regression run secondary-provider-smoke` 通过或给出预期 warning
- [ ] `novel regression run mixed-config-validation` 无混合配置问题

---

## 2. 数据库 backend 能力

- [ ] 能成功初始化 SQLite 项目
- [ ] 能成功初始化 MySQL 项目
- [ ] `config/database.json` 字段完整、backend 明确
- [ ] `novel doctor` 能展示当前 active backend
- [ ] `novel regression run database-backend-smoke` 能正确识别当前 backend
- [ ] `novel regression run sqlite-backend-smoke` 输出符合 SQLite 预期
- [ ] `novel regression run mysql-backend-smoke` 输出符合 MySQL 预期

---

## 3. 四条主链的阶段级可观测性

- [ ] planning 结果带有 `llmMetadata`
- [ ] generation 结果带有 `llmMetadata`
- [ ] review 结果带有 `llmMetadata`
- [ ] rewrite 结果带有 `llmMetadata`
- [ ] `chapter show <chapterId>` 可查看最新 plan/draft/review/rewrite 的元信息
- [ ] `doctor chapter <chapterId>` 可查看工作流链路与版本指针一致性

---

## 4. MySQL 读链兼容性

- [ ] `story show` 在 MySQL backend 下可正常执行
- [ ] `snapshot state` 在 MySQL backend 下可正常执行
- [ ] `snapshot chapter <chapterId>` 在 MySQL backend 下可正常执行
- [ ] `snapshot volume <volumeId>` 在 MySQL backend 下可正常执行
- [ ] `plan show / draft show / review show / rewrite show` 在 MySQL backend 下可正常执行
- [ ] `chapter show <chapterId>` 在 MySQL backend 下可正常执行

---

## 5. 卷级保护面

- [ ] `novel doctor volume <volumeId>` 可输出卷级风险摘要
- [ ] `novel regression volume <volumeId>` 可执行 volume suite
- [ ] `volume-plan-smoke / thread-progression-smoke / ending-readiness-smoke / volume-doctor-smoke` 已接入

---

## 6. 文档与入口

- [ ] `.env.example` 已包含 provider / timeout / retry / MySQL 示例
- [ ] `COMMAND_GUIDE.md` 已包含 provider 切换、阶段模型、SQLite/MySQL 初始化、doctor/regression 验收说明
- [ ] 本验收清单已提交到仓库

---

## 7. 推荐最终验收命令

```bash
npm run check -- --pretty false
novel doctor
novel regression list
novel regression run llm-provider-smoke
novel regression run secondary-provider-smoke
novel regression run mixed-config-validation
novel regression run database-backend-smoke
novel regression run sqlite-backend-smoke
novel regression run mysql-backend-smoke
```

如果项目已有卷：

```bash
novel doctor volume <volumeId>
novel regression volume <volumeId>
```
